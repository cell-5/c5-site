# cell5-vue-website

> Cell 5 Website

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:1234
npm start

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
